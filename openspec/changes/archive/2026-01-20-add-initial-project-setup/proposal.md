# Change: 添加初始项目基础框架

## Why
需要为 AI Farm 项目搭建初始的技术基础架构，包括前端静态页面、Python 后端 API、Supabase 数据库集成，以及支持 Netlify 单文件夹部署的配置。这是项目的第一个里程碑，为后续的功能开发提供基础。

## What Changes
- 创建静态前端框架（HTML、CSS、JavaScript）
- 搭建 Python 后端 API 基础结构（使用 Netlify Functions）
- 配置 Supabase 数据库连接
- 设置 Netlify 部署配置文件
- 创建项目目录结构，确保所有文件在单个文件夹内
- 添加环境变量配置示例
- 创建基础的 README 文档

## Impact
- 影响能力:
  - `frontend-static` - 新增静态前端能力
  - `backend-api` - 新增后端 API 能力
  - `database-integration` - 新增数据库集成能力
  - `deployment-config` - 新增部署配置能力
- 影响代码: 这是全新项目，将创建完整的项目结构
- 用户影响: 开发者可以基于此框架开始开发具体功能
